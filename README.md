🏢 ЖКХ: Система учета жильцов с льготами
📋 Описание проекта
Проект представляет собой десктопное приложение на C# (Windows Forms) для автоматизации учета жильцов с льготами в жилищно-коммунальном хозяйстве. Система позволяет вести базу данных жильцов, рассчитывать стоимость коммунальных услуг с учетом льготных скидок и выполнять различные операции с данными.

🎯 Основные возможности
👥 Управление жильцами
✅ Добавление новых жильцов с указанием льгот

✅ Редактирование существующих записей

✅ Удаление жильцов из системы

✅ Поиск и фильтрация данных

💰 Расчет стоимости услуг
⚡ Автоматический расчет стоимости с учетом скидок

🔢 Настройка базовой стоимости услуги

📊 Отображение итоговой суммы к оплате для каждого жильца

📊 Сортировка и фильтрация
🔍 Сортировка по фамилии, адресу или наличию льгот

📈 Группировка данных по различным критериям

💾 Работа с данными
💿 Подключение к MySQL базе данных

📁 Импорт/экспорт данных в формате CSV

🔄 Автономный режим работы (при отсутствии подключения к БД)

🛡️ Проверки и валидация
✅ Проверка корректности вводимых данных

⚠️ Контроль процентов скидки (0-100%, шаг 0.5%)

💵 Проверка стоимости услуг (0-100000)

📏 Ограничение длины полей ввода

🏗️ Архитектура проекта
Основные технологии
Язык: C# (.NET Framework / .NET Core)

Интерфейс: Windows Forms

База данных: MySQL
